# flask-blog
A blog site built with Flask

## Run Flaskblog 
To run this project locally, clone this repo and follow these steps:

install required packages
```bash
make install
```
run application
```bash
python3 run.py
```
## Setup Environment Variables & Run

```bash
export FLASK_APP=flaskblog.py
```
```bash
export FLASK_DEBUG=1
```
```bash
flask run
```
